# 凹语言吉祥物——鲵凹凹

![](./niwawa-20250613-01.jpg)

鲵凹凹是凹语言的第一个吉祥物，诞生时间大约在2022年的11月。鲵凹凹名字是从凹的谐音——娃娃鱼——而来，而娃娃鱼的学名大鲵。

在2025年初，凹语言社区同学设计了[WA酱吉祥物](https://wa-lang.org/smalltalk/st0069.html)。这是鲵凹凹和凹酱的互动照片：
